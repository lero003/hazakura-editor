Status: Draft for when a source tag is cut (not cut by this note alone)

Authority: Medium
Last reviewed: 2026-07-18

## Intent

Mark the in-tree **v2.0 Book Scope + Help** package as version **`2.0.0`**.

- No binary GitHub Release assets are required for a source-only tag.
- App Store upload / review / publication remain human-gated and separate.
- Do not move older tags (`v1.14.0`, `v1.13.0`, …).

## Included (source)

- Book Scope: select, app-private ordered groups, OKF-adapted suggestion draft,
  whole-book reader, PDF/EPUB export, saved-tree navigation, packaged internal
  links, preflight, UX quieting.
- OKF review + scaffold (existing contracts).
- In-app Help: **Books and knowledge folders**.
- Version metadata: npm, Tauri, Cargo, diagnostics About string → `2.0.0`.

## Not claimed until verified

- GitHub tag `v2.0.0` exists
- App Store package uploaded or published
- Signed TestFlight interaction matrix for the quieted UI
