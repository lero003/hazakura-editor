# Hazakura Editor 0.26.0 App Store Submission Candidate

> **Helper-free App Store lane package candidate**

Status: Local package candidate generated on 2026-06-20; later reported
as App Review-complete and released on the Mac App Store on 2026-06-20.
This note records local package evidence; raw App Store Connect,
TestFlight, and App Review logs are not tracked here.

`0.26.0` was prepared as the next App Store update after the reported
`0.25.0` App Store release and the `v0.26.0` source / local-app tag.
The App Store package was generated after the source tag so the package
build counter can advance without moving the source tag. Build `21`
supersedes build `20` with the Japanese e-book label corrected to
`電子書籍`.

The release boundary stays unchanged: Markdown source remains canonical,
Safe Editor remains primary, and the App Store lane omits Agent
Workbench, CLI Agent launch, dev mode, arbitrary command execution,
Apple Local Assist helper, external AI/API calls, Git, LSP, terminal,
plugins, workspace-wide indexing, auto-apply, and auto-commit behavior.
