# Hazakura Editor 0.31.0 App Store / TestFlight Candidate

> **Helper-enabled App Store lane package candidate**

Status: local package evidence generated on 2026-06-23 for TestFlight-side
v0.31 Reading Focus / Spread View smoke. App Store Connect upload,
processing, TestFlight install / launch, App Review, and release handling
are performed outside this repository unless separately recorded.

`0.31.0` packages the v0.31 reader work after the published `0.29.1`
App Store update. The release boundary stays unchanged: Markdown source
remains canonical, Safe Editor remains primary, and the App Store lane
omits Agent Workbench, CLI Agent launch, dev mode, arbitrary command
execution, external AI/API calls, Git, LSP, terminal, plugins,
workspace-wide indexing, auto-apply, and auto-commit behavior. Hazakura
Local Assist remains only an explicit, on-device writing companion whose
final result enters the existing unsaved edit transaction / Diff review
flow.

## v0.31 Candidate Focus

- e-book Mode keeps a book-like paged surface while wheel / trackpad /
  keyboard movement lowers page-turn friction.
- Spread View can show two pages when width allows and falls back to one
  page on narrow windows.
- `集中して読む` opens the same-window Reading Focus surface and returns
  through `編集に戻る`.
- Reading Focus and the side pane share e-book location state.
- Image-only Markdown paragraphs are promoted to dedicated
  `div.ebook-image-page` blocks so large images behave as full page
  units without forcing an extra nearly-empty simulated page.
