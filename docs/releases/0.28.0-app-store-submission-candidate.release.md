## v0.28 Release Focus

- L Mode image handling now follows Preview's external-image and
  embedded `data:image` safety policy.
- Workspace search now reuses the file-open decoding path for UTF-8,
  UTF-8 BOM, Shift-JIS, and EUC-JP text.
- External URL, Finder reveal, and print/browser handoff now share a
  fixed OS handoff boundary with static platform command templates.
- AI proposal review foundation is accepted as the existing explicit
  transaction / candidate Diff review path; broader file, paste, and
  multi-file ingest remain future work.
- Top chrome density is slightly tightened for multi-tab work: the tab
  item stays slim, the upper titlebar row gives the right-pane mode
  controls clearer vertical air, the mode controls read as a quieter
  single segment, and the mode order now reads from reading / structure
  toward diff / review. `確認` appears only when review changes are
  available.
