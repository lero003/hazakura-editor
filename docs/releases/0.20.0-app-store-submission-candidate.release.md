# Hazakura Editor 0.20.0 App Store Submission Candidate

> **Helper-free App Store lane package candidate**

Status: Local package candidate refreshed on 2026-06-19, then
superseded before upload by the `0.25.0` version alignment. Upload,
Apple validation, TestFlight distribution, manual TestFlight smoke, and
App Review are not claimed by this note.

Store approval and the `v0.20.0` Developer / GitHub warning-expected DMG

plugins, workspace-wide indexing, auto-apply, and auto-commit behavior.
