> **Helper-free App Store lane package candidate**

Status: Local package candidate generated on 2026-06-18. Upload, Apple
validation, TestFlight distribution, manual TestFlight smoke, and App
Review are not claimed by this note.

Store approval and the `v0.20.0` Developer / GitHub warning-expected DMG
