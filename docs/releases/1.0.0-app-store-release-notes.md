# Hazakura Editor 1.0.0 App Store Release Notes

Status: Local Candidate Prepared
Scope: User-facing App Store / TestFlight release-note copy for `1.0.0`
Authority: Medium
Last reviewed: 2026-06-28

This note prepares the `1.0.0` release message. A signed local candidate
has passed the documented local release gates. It does not claim App
Store Connect upload, TestFlight distribution, App Review approval, or
public release.

## App Store Connect Copy

Hazakura Editor v1.0は、Markdownで書く、本として読む、ローカルAIで整える体験をひとつにまとめた最初の正式版です。

・L ModeとMarkdownプレビューで、sourceを保ったまま執筆できます。
・電子書籍モードと見開き表示で、長い原稿を本のように読み返せます。
・読んで気づいた場所から、対応するMarkdownへ戻って修正できます。
・Hazakura Local Assistの提案をDiffで確認し、採用または破棄できます。
・MarkdownをEPUB、PDF、HTMLへ明示的に書き出せます。
・ワークスペース、保存、Recovery、長文表示の安定性を整えました。

## Release Position

`1.0.0` is a semantic and communication re-baseline of the feature
shape already shipped through App Store version `0.36.0`. It does not
introduce a second document model or a new AI execution surface.

The product promise is:

> Markdownで書き、本として読み、ローカルAIで整える。

The core flow is write -> read -> notice -> return -> revise -> review ->
export. Markdown source remains canonical throughout that flow.

## Boundary

Hazakura Local Assist remains a preview, availability-gated on-device
writing companion. It has no external AI/API fallback, background
rewriting, auto-save, tool calling, or workspace-wide indexing. Its
proposals remain explicit, unsaved, and Diff / Discard reviewable.

EPUB export remains an explicit single-document workflow. `1.0.0` does
not claim advanced metadata, cover design, navigation editing, multi-file
book management, or a full EPUB production environment.

The App Store lane excludes Agent Workbench, CLI Agent launch, arbitrary
command execution, external AI/API calls, and provider-add UI.

## Verification Boundary

Local source, build, audit, signing, entitlement, package, checksum, and
sandbox-preview gates passed for the prepared local candidate. The
Rust audit reported only the previously triaged transitive Tauri / Linux
GTK and related unmaintained warnings; no vulnerability finding blocked
this macOS candidate.
Golden Manuscript product evidence may be recorded against the published
`0.36.0` build or the same-code `1.0.0` TestFlight candidate. Before
public `1.0.0` release, TestFlight still needs a short install / launch,
Open / Save, e-book Mode, Local Assist, EPUB, and PDF regression pass.
