# 夜光の図書室

## 第一章　午前四時の来訪者

雨の音だけが聞こえる深夜の図書室で、私はひとりの来訪者を待っていた。

午前四時。窓の外はまだ完全に暗いけれど、空の端がほんの僅かに、夜明けの予兆のような青みを帯び始めている時刻だ。机の上には冷めきったコーヒーと、読みかけの原稿用紙が散らばっている。原稿のタイトルは「夜光の図書室」。書き始めてから半年になるが、まだ終わりが見えない。

> 執筆とは、暗闇のなかで手探りで形を探す作業に似ている。言葉という輪郭を、少しずつ、慎重になぞっていく。

来訪者は約束の時間より三十分遅れて現れた。濡れた傘を入り口に立てかけ、静かに私の向かいに座る。その人物が誰なのか、私は知らない。ただ、この図書室に「深夜の相談相手」として紹介されてやって来たことだけは聞かされていた。

「原稿、読ませてもらいました」

来訪者はそう言って、散らばった用紙を一瞥した。

## 第三章　図書室の朝

夜が明け始めた。窓ガラス越しに、図書室の棚に並んだ本の背表紙が、朝の光を反射して淡く光っている。

```text
06:12 — 最初の光が本棚に届く
06:24 — 雨が上がる
06:41 — 来訪者が帰る
```

私は机の上を片付け、原稿を一つのファイルにまとめた。図書室の静けさが、書き終えた物語をそっと受け入れてくれたような気がした。
