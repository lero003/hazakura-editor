> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Restore readable fenced code blocks and complete Markdown tables in v1.3 A4 PDF exports without replacing the direct WebKit/PDFKit export architecture or removing margin presets.

**Architecture:** Keep the existing one-A4-high horizontal CSS column layout and native capture path. Restore the v1.2 code-block contrast, annotate each rendered table with its column count, and replace native table fragmentation only inside export HTML with row-level CSS grid layout that can continue between A4 columns.

**Tech Stack:** React 19, TypeScript, Vitest/Testing Library, CSS multi-column layout, Tauri 2, WebKit PDF capture, Poppler rendering.

### Task 2: Restore PDF-Safe Code And Table Layout

**Files:**
- Modify: `src/features/document/pdfExport.ts`
- Modify: `src/hooks/document/useDocumentExport.ts`
- Test: `src/features/document/pdfExport.test.ts`
- Test: `src/hooks/document/useDocumentExport.test.tsx`

- [ ] **Step 1: Implement the minimal PDF-only cascade**

Keep `break-inside: avoid` for code blocks, blockquotes, and images. Annotate each exported table with `--pdf-table-columns`, then render `table` / `thead` / `tbody` as blocks and each `tr` as a fixed-column grid so rows remain readable and may continue across A4 columns. Add screen-capture overrides after the shared preview CSS:

```css
.markdown-preview pre {
  background: var(--status-bg);
  border-left: 2px solid #999999;
  box-sizing: border-box;
  color: var(--status-text);
  display: inline-block;
  max-width: 100%;
  overflow: visible;
  padding: 16px;
  white-space: pre-wrap;
  width: 100%;
}
.markdown-preview pre code {
  background: transparent;
  color: inherit;
}
.markdown-preview .markdown-table-frame {
  break-inside: auto;
  max-width: 100%;
  overflow: visible;
}
.markdown-preview .markdown-table-frame table {
  display: block;
  min-width: 0;
  width: 100%;
}
.markdown-preview .markdown-table-frame thead,
.markdown-preview .markdown-table-frame tbody {
  display: block;
}
.markdown-preview .markdown-table-frame tr {
  break-inside: avoid;
  display: grid;
  grid-template-columns: repeat(var(--pdf-table-columns), minmax(0, 1fr));
}
.markdown-preview .markdown-table-frame th,
.markdown-preview .markdown-table-frame td {
  box-sizing: border-box;
  min-width: 0;
  overflow-wrap: anywhere;
  white-space: normal;
}
```

Do not change `src-tauri/src/commands/export.rs`, page geometry, security validation, margin values, or the non-PDF preview stylesheet.
