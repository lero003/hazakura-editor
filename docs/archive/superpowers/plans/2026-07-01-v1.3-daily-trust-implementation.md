# v1.3 Daily Trust Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement four independent v1.3 trust slices: Save As session continuity, explicit Local Assist accept/discard, richer e-book TOC context, and A4 PDF margin presets.

**Architecture:** Keep each slice behind its existing owner. `AppWorkspace` continues to own document view state, `AppleAssistReviewBar` owns AI transaction review, `EBookPane` derives display metadata from the existing chapter parser, and `useDocumentExport` owns explicit export requests while a focused PDF feature module owns allowlisted preset data.

**Tech Stack:** React 19, TypeScript, CodeMirror 6, Vitest/Testing Library, Tauri 2, Rust/WebKit PDF export, CSS.

Add this helper:

```ts
export function rekeyDocumentViewState(
  current: DocumentViewStateRegistry,
  previousKey: string,
  nextKey: string,
): DocumentViewStateRegistry {
  if (previousKey === nextKey || current[previousKey] === undefined) {
    return current;
  }
  const { [previousKey]: previous, ...rest } = current;
  return {
    ...rest,
    [nextKey]: { ...(rest[nextKey] ?? {}), ...previous },
  };
}
```

Add:

```ts
export function collectEbookChapterSubheadings(
  rawChapters: EbookChapter[],
  topLevelChapters: EbookChapter[],
): string[][] {
  return topLevelChapters.map((chapter, index) => {
    const nextStart = topLevelChapters[index + 1]?.startLine ?? Infinity;
    return rawChapters
      .filter((candidate) =>
        candidate.startLine > chapter.startLine &&
        candidate.startLine < nextStart &&
        (candidate.headingLevel ?? 0) >= 3 &&
        candidate.headingText,
      )
      .map((candidate) => candidate.headingText as string);
  });
}
```

- [ ] **Step 3: Implement the allowlisted model**

```ts
export type PdfMarginPreset = "narrow" | "standard" | "wide";

export const DEFAULT_PDF_MARGIN_PRESET: PdfMarginPreset = "standard";

export const PDF_MARGIN_PRESETS = {
  narrow: { blockMm: 10, inlineMm: 10 },
  standard: { blockMm: 18, inlineMm: 16 },
  wide: { blockMm: 25, inlineMm: 22 },
} as const satisfies Record<PdfMarginPreset, {
  blockMm: number;
  inlineMm: number;
}>;

export function pdfMarginCss(preset: PdfMarginPreset): string {
  const value = PDF_MARGIN_PRESETS[preset];
  return `${value.blockMm}mm ${value.inlineMm}mm`;
}
```
